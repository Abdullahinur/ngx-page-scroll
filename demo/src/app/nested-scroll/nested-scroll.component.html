Nested scrolling

Showcasing the scrolling inside a custom container. Triggering is done using the scroll service. In the complex card the scroll target is nested in multiple parents with position:relative. This prevents the basic usage of offsetTop to determine the element's location. Therefore the advancedInlineOffsetCalculation is used for that example to attain the correct scroll position.
