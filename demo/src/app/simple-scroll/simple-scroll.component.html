Simple Scrolling

On this page you may find some basic scroll operations using the directive and using the service.

All the following buttons scroll to the seventh heading but with different configuration.

Non-Interruptible, custom duration

This button has a custom animation duration of 5 seconds and is specified to be non-interrupbtible. Try scrolling or using the arrow keys while the scroll animation takes place.

Target Offset

These two buttons trigger a scroll animation with a custom offset to the target. One will stop 150px atop the target, the second one 50px below the target.

Scroll target with router to current page

The following scroll animation refers to a scroll target on the current page and includes the routerLink directive pointing to the current page. Each button uses a different routerLink syntax.

Finish event

The following scroll animation has a custom duration of 350ms and a pageScrollFinish event listener attached and shows a snackbar at the bottom of the page when the event fires.

Custom easing

You may specify a custom easing function to manipulate the scroll position over time. This example uses an easeInOutExpo function.

Scrolling using PageScrollService

You may use the PageScrollService to create highly customized scroll animations. This example will scroll to the last heading of this page.
